# DIGI-BOW

An advanced, integrated hardware and software engineering system designed for the precise measurement, tracking, and analysis of human jaw movements. 

DIGI-BOW was awarded 1st Place at the prestigious IPS PG 2026 competition for its innovative approach to biomedical and dental engineering instrumentation.

---

## Project Overview

Accurate tracking of mandibular (jaw) movement is critical for diagnostics, temporomandibular joint (TMJ) disorders, and prosthodontic treatments. DIGI-BOW bridges the gap between mechanical dental architecture and digital precision. 

The system utilizes a custom-engineered hardware apparatus integrated with microcontrollers and sensors to capture real-time spatial data, which is then processed and visualized through a dedicated software dashboard.

### Key Features
* Real-Time Kinematic Tracking: High-precision data acquisition of complex jaw trajectories.
* Hardware-Software Integration: Seamless low-latency communication between custom sensor arrays and the analytical web dashboard.
* Clinical Diagnostic Utility: Designed to provide data-driven insights for dental professionals and researchers.

## System Architecture (High-Level)

The system is split into two primary segments:
1. Hardware Module: A lightweight, non-invasive wearable assembly featuring microcontrollers and integrated sensors calibrated for fine-grained physical displacement.
2. Software Dashboard: A localized telemetry interface that ingests sensor data streams, applies filtering algorithms, and plots real-time movement graphs for clinical evaluation.
